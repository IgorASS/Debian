O que é o Debian

Debian, é um sistema operacional composto inteiramente de software livre. O Debian foi lançado em 16 Agosto de 1993 por Ian Murdock, ao tempo estudante universitário, que escreveu o Manifesto Debian que apelava à criação de uma distribuição GNU/Linux a ser mantida de uma maneira livre, segundo o espírito do GNU.

Para que serve o Debian

O Debian é especialmente conhecido pelo seu sistema de gestão de pacotes, chamado APT, que permite: atualização relativamente fácil a partir de versões relativamente antigas; instalação quase sem esforço para novos pacotes e remoção limpa de pacotes antigos.

Exemplos de utilização do Debian

O Debian vem com mais de 59000 programas diferentes. Cada bit dele é livre. Se você tem software privativo que funciona no GNU/Linux ou GNU/kFreeBSD, você ainda pode usá-lo - na verdade, pode haver até um instalador para ele no Debian que irá instalá-lo e configurá-lo automaticamente para você.
